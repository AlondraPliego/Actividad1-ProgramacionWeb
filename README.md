# Actividad 1-Programacion Web
La actividad presentada es la realización de ejercicios con los componentes básicos de HTML, como se utilizan las hojas de estilo CSS y la implementación de JS como un lenguaje que nos permitirá realizar acciones relevantes, teniendo como objetivo adquirir los conocimientos necesarios para la introducción a la creación de una aplición web.
## Ejercicios realizados
1. **Ejercicio 1:** Hacemos el clásico hola mundo para entender la estructura básica de HTML5.
2. **Ejercicio 2:** Haciendo uso de etiquetas simples de diseño, le damos al texto presentaciones diferentes. Además de hacer uso de parrafos.
3. **Ejercicio 3:** Comenzamos a utilizar enlaces externos y el como son cargados.
4. **Ejercicio 4:** Insertar imagenes con diferentes atributos.
5. **Ejercicio 5:** Integración de los 4 ejercicios anteriores.
6. **Ejercicio 6:** Se presentan las secciones semánticas básicas con las que podemos trabajar.
7. **Ejercicio 7:** Hacemos uso de listas anidadas.
8. **Ejercicio 8:** Diseño de un horario semanal utilizando tablas y su correspondiente estructura. Además de añadir codigo CSS en una determinada hoja de estilos.
9. **Ejercicio 9:** Se realiza un formulario de contacto con el fin de utilizar validaciones y CSS externo.
10. **Ejercicio 10:** Conversión de °C a °F, comenzamos a hacer uso básico de JS, para la realización de operaciones mátematicas.
11. **Ejercicio 11:** Conversión de km a millas, utilizando restricciones como no poder utilizar numeros menores a 0.
12. **Ejercicio 12:** Conversión de pesos mexicanos a dolares.
13. **Ejercicio 13:** Uso de condiciones en JS para determinar si una persona de determinado rango de edad puede votar o no.
14. **Ejercicio 14:** Inicio de uso de arreglos básicos en JS, determinar un numero mayor, menor y promedio.
15. **Ejercicio 15:** Utilización de arreglos en JS para calcular y buscar el nombre del estudiante, la calificación más alta, baja y el promedio general.
